Video Factory

Python MoviePy Yt-dlp Edge-TTS Gemini API Google Drive API

เกี่ยวกับโปรเจกต์ (Overview)

Video Factory คือโปรเจกต์ส่วนตัวสำหรับสร้างคลิปสั้น (Shorts/TikTok) แบบอัตโนมัติ โดยใช้ AI ช่วยตั้งแต่การดูเนื้อหา เขียนบท พากย์เสียง และตัดต่อ เพียงแค่วางลิงก์ YouTube ที่ต้องการ ระบบจะทำการสร้างคลิปพร้อมเสียงพากย์และอวตารให้ทันที

กระบวนการทำงาน (Workflow)

  • 1. Download Video: ใช้ yt-dlp ดาวน์โหลดวิดีโอต้นฉบับมาเก็บไว้
  • 2. Content Generation: ใช้ Gemini AI ดูเนื้อหาในคลิปและ "เขียนบทพากย์" ขึ้นมาใหม่ให้น่าสนใจ
  • 3. Text-to-Speech: ใช้ Edge TTS แปลงบทเป็นเสียงพากย์ที่ลื่นไหล
  • 4. Auto Editing: ใช้ MoviePy นำคลิป เสียงพากย์ และรูปอวตารมาตัดต่อรวมกันโดยอัตโนมัติ
  • 5. Cloud Upload: อัปโหลดผลงานที่เสร็จสมบูรณ์ขึ้น Google Drive ผ่าน API พร้อมใช้งานทันที

บทบาทและความรับผิดชอบ (Role)

รับผิดชอบโปรเจกต์ทั้งหมด (Solo Developer) ตั้งแต่การวางแผน Architecture, เขียน Script ตัดต่อ Python และเชื่อมต่อ API ต่างๆ
