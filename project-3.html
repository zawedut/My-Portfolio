Memo Bot

Tech Stack: Next.js, Python, MoviePy, Yt-dlp, edge-tts, Gemini API, Google Drive API

Video Factory คือโปรเจ็คส่วนตัวที่ทำมาเพื่อใช้งานในการสร้างคลิปสั้นเพื่อโพสลงใน tiktok โดยได้ใช้ AI ในการพากย์เสียงเเละการใช้ Python ในการตัดต่อวิดิโอให้อัตโนมัติ เเละใช้งานโดยการเพียงวางลิงก์ youtube ที่ต้องการจำพากย์คลิปนั้น Gemini จะทำการดูเนื้อหาเเละสร้างข้อความพากย์ขึ้นมาเเละ edge-tss นำมาพากย์เเละนำเสียงกับตัวอวตารที่เราทำเอาไว้รวมถึงคลิปมารวมกันด้วย MoviePy เเละเก็บคลิปที่สร้างเสร็จขึ้นไว้บน Google Drive


1.Download Video เราใช้ yt-dlp ในการดาวน์โหลดวิดีโอจาก YouTube เเละเก็บไว้ในโฟลเดอร์ที่กำหนด

2.Gemini AI ใช้ในการสร้างข้อความพากย์จากเนื้อหาของวิดีโอ

3.Edge TTS ใช้ในการแปลงข้อความพากย์เป็นเสียง

4.MoviePy ใช้ในการตัดต่อวิดีโอและใส่เสียงพากย์เข้าไปในวิดีโอ

5.Google Drive API ใช้ในการอัปโหลดวิดีโอที่ตัดต่อเสร็จแล้วไปยัง Google Drive
