KU Food Araidee (กินไรดี)

Next.js TypeScript FastAPI Python Scikit-learn OpenTyphoon API PostgreSQL

เกี่ยวกับโปรเจกต์ (Overview)

KU Food Araidee คือแอปแนะนำอาหารสำหรับนิสิตและบุคลากร ม.เกษตรศาสตร์ ที่ใช้ AI เรียนรู้รสนิยมส่วนตัวและนำเสนออาหารผ่านการปัดซ้าย-ขวา สไตล์ Tinder พัฒนาเพื่อแข่งขันใน KU AI Pioneers: Forward Challenge และผ่านเข้ารอบ 54 ทีมสุดท้าย

ฟีเจอร์หลัก (Key Features)

  • AI Food Recommendation: ระบบ AI ปรับตัวตามพฤติกรรมการกิน — ใช้ Typhoon LLM สำหรับผู้ใช้ใหม่ และ KNN สำหรับผู้ใช้ที่มีประวัติเพียงพอ
  • Swipe Interface: ปัดซ้าย (ไม่เอา) หรือขวา (เอา) เหมือน Dating App ทำให้การเลือกอาหารสนุกและรวดเร็ว
  • Smart Filtering: กรองตามงบประมาณและตัดเมนูที่เคยกินออก เพื่อลดความซ้ำซาก
  • KU Exclusive: ฐานข้อมูลร้านอาหารใน ม.เกษตรศาสตร์ โดยเฉพาะ

บทบาทและความรับผิดชอบ (Role)

รับผิดชอบด้าน AI Development ทั้งหมด ตั้งแต่การออกแบบระบบ Recommendation Engine, การพัฒนา KNN Model และการเชื่อมต่อ OpenTyphoon API
