Hello Adventurer by KU JETH

Next.js Gemini API Generative AI Web RPG

เกี่ยวกับโปรเจกต์ (Overview)

Survive.exe: The First Jobber คือ Web-based RPG Game สไตล์ Pixel ที่ผู้เล่นรับบทเป็นนักผจญภัยหางาน และต้องแยกแยะระหว่างงานจริงกับ Job Scam ผ่านการแชทกับ NPC ที่ขับเคลื่อนด้วย Gemini AI พัฒนาโดยทีม KU JETH เพื่อแข่งขันใน Google Securing Digital Trust: Thailand Anti-Scam Ideathon 2026 โดยผ่านเข้ารอบ 20 ทีมสุดท้าย

ฟีเจอร์หลัก (Key Features)

  • AI Dynamic NPC: ใช้ Gemini API สร้างบทสนทนาผู้ว่าจ้าง — งานจริงตอบชัดเจน, Scam เลี่ยงคำถามและเร่งตัดสินใจ
  • Choice-based Gameplay: ผู้เล่นแชทต่อรอง ถามรายละเอียด และตัดสินใจรับหรือปฏิเสธภารกิจ
  • Adaptive Difficulty: AI ปรับความซับซ้อนของกลลวงให้ยากขึ้นตามความสามารถของผู้เล่น
  • Safe-to-Fail: เรียนรู้การสังเกต Job Scam โดยไม่ต้องเสียเงินจริง — เพียง Game Over

บทบาทและความรับผิดชอบ (Role)

รับผิดชอบด้าน Fullstack Developer — พัฒนาระบบ Gameplay, การเชื่อมต่อ Gemini API และ UI ของเกม
