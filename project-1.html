Mind Buddy

Next.JS TypeScript Prisma PostgreSQL Typhoon API Line Messaging API

เกี่ยวกับโปรเจกต์ (Overview)

Mind Buddy คือโปรเจกต์ Line OA ที่พัฒนาขึ้นเพื่อแข่งขันในงาน KU Innovation 5 ซึ่งจัดร่วมกันระหว่างมหาวิทยาลัยเกษตรศาสตร์และกรุงเทพมหานคร โดยมุ่งเน้นการดูแลและส่งเสริมสุขภาพจิตของนิสิตผ่านฟีเจอร์ต่างๆ ที่เข้าถึงง่ายบนแพลตฟอร์ม LINE

ฟีเจอร์หลัก (Key Features)

  • 1. ปฏิทินอารมณ์ (Mood Calendar): บันทึกอารมณ์ในแต่ละวัน หากพบแนวโน้มอารมณ์เชิงลบ AI Chatbot จะทักไปพูดคุยและให้กำลังใจทันที
  • 2. Buddy Review: เว็บไซต์รีวิวคณะและมหาวิทยาลัยต่างๆ โดยนิสิตปัจจุบัน เพื่อเป็นข้อมูลให้น้องๆ มัธยม
  • 3. Learn With Buddy & Mind Care: ศูนย์รวมแหล่งเรียนรู้ ค่ายกิจกรรม และรายชื่อคลินิก/โรงพยาบาลด้านสุขภาพจิต
  • 4. AI Chatbot: ใช้ Typhoon API (LLM ภาษาไทย) ในการโต้ตอบและให้ข้อมูลเกี่ยวกับบริการต่างๆ ของ Mind Buddy ได้อย่างเป็นธรรมชาติ

บทบาทและความรับผิดชอบ (Role & Responsibility)

รับผิดชอบงานด้าน Technical ทั้งหมดของทีม ตั้งแต่การออกแบบระบบ (System Design), การพัฒนา Line OA, การเชื่อมต่อ API, การพัฒนา Web Application ทั้งฝั่ง Frontend และ Backend รวมถึงการจัดการฐานข้อมูล
